## Resumo

A plataforma [CulturaEduca.cc](https://plataforma.culturaeduca.cc/) atua como um Sistema de Informações Geográficas participativo voltado à territorialização de políticas públicas e intersetorialidade entre equipamentos diversos. Entretanto, a extração de insights territoriais complexos na plataforma ainda é limitada pela rigidez de bancos de dados relacionais para consultas multicamada.

Este trabalho propõe a migração de dados geoespaciais para uma arquitetura baseada em Bancos de Dados em Grafos integrada a Modelos de Linguagem de Grande Escala. A metodologia compreende a modelagem ontológica do território, a implementação de uma interface Text-to-Cypher via sistemas multi-agentes e a validação técnica por meio de métricas de acurácia de consultas e usabilidade com usuários finais.
